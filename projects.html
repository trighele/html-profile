Kubernetes Migration Project


I successfully led a 10-month project to migrate a critical application, consisting of 17 microservices and 6 frontends, from Azure Virtual Machines to a Kubernetes environment. The project was completed without any issues post go-live, enhancing the application's scalability and efficiency.

Project Short Squeeze


Project Short Squeeze is a powerful tool designed to screen and compile data on potential short squeeze opportunities. By collecting and analyzing key metrics such as short interest, trading volume, and price movements, the project helps users identify stocks that may experience rapid price increases due to short squeezes. The tool stores this data in a database, enabling thorough analysis and informed investment decisions. The application is containerized using Docker and Docker Compose for easy setup and deployment. Craigslist and Facebook Marketplace Scraper


I developed a Python-based tool to automatically scrape Craigslist and Facebook Marketplace for new items matching specific criteria. This tool runs every 15 minutes, leveraging the asynchronous capabilities of the Playwright and HTTPX modules for efficient and fast data gathering. The results are meticulously filtered to ensure only new and relevant items are identified and sent to a specified email address, providing users with timely notifications of potential opportunities. The link to the code can be found here on GitHub.
